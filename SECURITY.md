# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

1. **Do not open a public issue**
2. Email security@ainfera.ai with details and reproduction steps
3. You will receive a response within 48 hours
4. Do not disclose the vulnerability publicly until a fix is released

## Security Measures

- All inference responses carry cryptographically signed audit receipts
- Audit chain can be independently verified offline using [ainfera-verify](https://github.com/ainfera-ai/verify)
- No trusted third parties required for verification
- AgentCards are signed and tamper-evident

## Supported Versions

Only the latest release receives security updates.
